## html 기본

    content에 tag를 앞뒤로 붙힘.
    해당 content가 무엇인지를 브라우저가 알수 있게 하는 것.

```html
<tag> content </tag>
```

반드시 태그를 열고 닫아야함 <br>
태그 안에 content를 씀

### tag 종류

#### h tag

```html
    <h1>Hi! kimchiman!</h1>
    <h2>Hi! kimchiman!</h1>
    <h3>Hi! kimchiman!</h1>
    <h4>Hi! kimchiman!</h1>
    <h5>Hi! kimchiman!</h1>
    <h6>Hi! kimchiman!</h1>
```

제목을 사용할때 사용

#### ul tag

```html
<ul></ul>
```

ul 태그는 (unodered list) 순서가 없는 리스트

#### li tag

```html
<ul>
  <li>Beer</li>
  <li>asdf</li>
  <li>asd</li>
  <li>aa</li>
</ul>
```

li 태그는 (list item) 리스트 안 아이템

#### ol tag

```html
<ol></ol>
```

ul 태그는 (odered list) 순서가 있는 리스트
